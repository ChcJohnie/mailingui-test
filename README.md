## Getting Started

Standard next steps and commands apply

To get sending with integrations working .env.local have to be added

```env
# Resend integration
RESEND_API_KEY=
RESEND_SENDER=

# Nodemailer integration
NODEMAILER_HOST=
NODEMAILER_USER=
NODEMAILER_PASS=
NODEMAILER_SENDER=
```
